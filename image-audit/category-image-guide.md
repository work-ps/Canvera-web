## How to Use This Guide

Each table below lists every image the website needs, organized by the page or section where it appears. For each image you will find the recommended size (width x height in pixels), the aspect ratio, and notes on what the photo should show.

**Key terms:**
- **WebP** — Modern image format, smaller files, great quality. Preferred.
- **JPG** — Standard photo format. Good fallback.
- **PNG** — Use only when transparency is needed (logos, icons).
- **SVG** — Vector format for icons and logos. Infinitely scalable.
- **@2x** — Double the displayed size, so images look crisp on retina screens.
- **Aspect ratio** — The shape of the image (e.g. 4:3 is slightly wider than tall, 1:1 is square).

## Photography Guidelines

### Product Photos
- **Background:** Clean white or very light grey (#FAFAFA)
- **Lighting:** Soft, even studio lighting. No harsh shadows.
- **Angle:** 3/4 view showing both cover face and spine thickness
- **Props:** Minimal — maybe a subtle surface texture (marble, wood) beneath
- **Consistency:** All 25 products should have the same lighting setup, angle, and background

### Lifestyle / Hero Photos
- **Style:** Warm, aspirational, professional
- **Subjects:** Albums in real contexts — on a coffee table, in a photographer's hands, at a wedding venue
- **Color palette:** Align with brand tones (petrol teal, warm amber, clean neutrals)

### Collection Images
- **Focus:** Emphasize the material/texture that defines each collection
- **Detail:** Close enough to see material quality (leather grain, wood texture, suede nap, foil shine)
- **Lighting:** Dramatic lighting to highlight textures

### File Format Requirements
- **Delivery format:** High-res originals in TIFF or PNG (lossless)
- **Web format (we convert):** WebP primary, JPG fallback
- **Color space:** sRGB
- **Resolution:** Provide at @2x sizes listed above (the app will downscale for standard screens)
